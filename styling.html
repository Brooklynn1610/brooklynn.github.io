My CSS Approach

While Bootstrap 5 provides an excellent foundation for responsive layout, I used custom CSS (custom.css) to establish my personal brand identity.

Key Customizations:

  • Image Styling: I created a `.profile-img` class to ensure my headshot is perfectly circular, properly scaled, and features a subtle drop shadow to lift it off the background.
  • Card Hover Effects: I added CSS transition properties to the `.card` elements on the portfolio page. When a user hovers over a project, it smoothly translates upwards on the Y-axis, providing interactive feedback.
  • Typography & Color: I overrode Bootstrap's default fonts with a modern sans-serif stack to make the resume easily scannable, and adjusted the background colors of the hero section for better contrast.
